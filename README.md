# KoTR_ClutchMod
Hard Truck 2/King of the Road 8.2 asi plugin, which adds support of steering wheels with clutch and gearbox.
Плагин для игры "Дальнобойщики 2" версии 8.2, позволяющий использовать рули со сцеплением и коробкой передач.

Файл "CPatch.h" взят из проекта [D2DInputWrapper](https://github.com/Voron295/rignroll-dinput-wrapper) от Voron295.
